Clip Hist - Windows 使用教程

快速上手剪贴板历史，支持全局快捷键、托盘图标、自动隐藏与便捷操作。

快速开始 基本操作 历史与动作 设置说明 托盘与窗口 构建与打包 常见问题

快速开始

系统要求

  • Windows 10 或更高版本（x64）
  • 若需本地构建：Node.js 18+、Rust Stable、Tauri 相关依赖

开发运行

npm install
npm run tauri dev

打包发布

npm run tauri build

基本操作

  • 打开/隐藏窗口：
    • 默认全局快捷键：Ctrl + D（可在设置中修改）
    • 托盘图标：左键单击在鼠标所在屏幕显示/隐藏窗口；右键打开菜单（显示历史、退出）
    • 主界面按 Esc 立即隐藏窗口
  • 自动隐藏：点击窗口外部区域时，窗口会自动隐藏
  • 拖动窗口：将鼠标置于窗口顶部空白拖拽区域按住左键拖动
若左键打开出现闪烁，已在程序内部处理释放时触发与短暂抑制隐藏，确保单击稳定打开。

历史与动作

  • 浏览历史：滚动查看剪贴板记录，支持搜索
  • 复制到剪贴板：选中条目后按 Enter 或点击复制
  • 动作菜单：
    • 打开：Ctrl + K 或界面底部 “Actions”
    • 支持：复制、编辑、删除
    • 编辑对话框：Esc 取消，Ctrl + Enter 保存

设置说明

  • 外观主题：亮色 / 暗色 / 跟随系统
  • 主操作：可选择复制等默认行为
  • 历史保留时间：1天 / 7天 / 30天 / 3个月 / 1年 / 永久
    • 在应用启动时会自动清理早于设定时限的历史记录
  • 链接预览：可开关
  • 开机自启动：可通过设置开启/关闭
  • 全局快捷键：点击录制按钮，按下想要的组合键保存（如 Ctrl + Shift + D）

托盘与窗口

  • 托盘图标位于任务栏右下角通知区域
  • 左键单击：在鼠标所在的显示器边缘弹出窗口，并自动聚焦
  • 右键菜单：显示历史、退出应用
  • 窗口在失去焦点或点击外部区域时自动隐藏

构建与打包

  • 开发调试：npm run tauri dev
  • 正式打包：npm run tauri build（生成安装包/可执行文件）

常见问题

1. 全局快捷键与其他程序冲突

打开设置 → 全局快捷键，点击录制，输入新的组合键后保存（如 Ctrl + Shift + Space）。

2. 单击托盘图标后窗口闪一下

已优化为在鼠标释放（Up）时执行切换，并增加短暂“隐藏抑制”，确保窗口稳定显示。

3. 启动后历史记录变少

请在设置中检查“历史保留时间”。应用启动时会清理超出保留期的历史记录。

4. 开机自启动未生效

请在设置中确认已开启“开机自启动”。若仍无效，可尝试关闭后重新开启。

本文适用于 Windows 版本。其他平台可能在快捷键与托盘行为上有所不同。
