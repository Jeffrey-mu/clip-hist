# 更新日志 (Changelog)

本项目所有的重大改动、功能添加和修复都将记录在此文件中。

## [未发布] (Unreleased)

### ✨ 新特性 (Features)
- **系统托盘**: 托盘菜单本地化为中文，并新增“设置”选项。
- **快捷键**: 新增全局快捷键自定义功能，支持在设置面板中修改并持久化保存。
- **剪贴板监听**: 支持多颜色格式检测（如 `#RRGGBB`, `rgb()`, `hsl()` 等），提取真实的颜色值进行展示，并在需要时保持原始格式。
- **极速粘贴**: 彻底重构粘贴到其他应用的逻辑，引入 `enigo` 库模拟原生操作系统的键盘事件，替代原有的 PowerShell 和 AppleScript 子进程方案，实现近乎 **0 延迟** 的粘贴体验。

### 🐛 修复 (Bug Fixes)
- **终端闪烁**: 修复了由于剪贴板文本比对逻辑错误导致控制台持续输出和闪烁的问题，加入了对非文本内容的检测并跳过。

### 🚀 性能优化 (Performance)
- **后端剪贴板监听**: 在 Windows 平台上引入 `clipboard_win::raw::seq_num()` 获取系统剪贴板序列号，避免了后台空闲时高频度读取图片数据带来的 CPU 和内存浪费。
- **数据库查询**: 为 SQLite 数据库 `clipboard_history` 表的 `created_at` 字段添加索引，极大提升了大量历史记录下的分页查询和加载速度。
- **前端渲染**: 提取了 `PreviewPane.tsx` 和历史列表组件中的正则表达式及长数组配置常量到模块作用域，避免组件重渲染时产生的重复实例化开销。

### 💄 UI/UX (UI & UX)
- **颜色展示优化**: 优化剪贴板中颜色格式条目的预览，将颜色块从带阴影的方形修改为带粗边框的圆形，自动适配深色和浅色模式，提升整体视觉体验。
